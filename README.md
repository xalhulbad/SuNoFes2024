# Happily Ever After  

**Happily Ever After** is a narrative-driven visual novel based on the player's decisions, featuring:  
- 🎭 **10 unique endings**  
- 🎨 **Custom artwork**  
- 🎮 **Dynamic GUI**

Every choice you make shapes the story's outcome.
